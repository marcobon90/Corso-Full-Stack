Guida Sass

SASS è uno dei più famosi e usati preprocessori CSS.

Installazione

E' possibile usare SASS semplicemente installando questa estensione di VS Code

Dopo l'installazione apparirà la scritta Wathc Sass nella riga in basso dell'editor. Cliccandoci si attiverà l'estensione che leggerà tutti i file con estensione .scss e li compilerà in file CSS da importare nelle pagine web.
